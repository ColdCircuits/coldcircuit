# ColdCircuit Branding Guide

## Brand Identity

**Name:** ColdCircuit

**Tagline:** AI-managed crypto, fully transparent.

**Twitter/X Handle:** @AICryptoTrades

**Domain:** coldcircuit.io (or GitHub Pages URL)

---

## Brand Concept

"ColdCircuit" evokes:
- **Cold** - Emotionless, algorithmic, disciplined (vs emotional human trading)
- **Circuit** - Automated, systematic, electrical/computational
- Tech-forward, serious, professional

The name suggests a machine running without emotion - perfect for an AI trading bot.

---

## Tone & Voice

- **Professional but accessible** - No jargon walls
- **Transparent, not promotional** - "Here's what happened and why"
- **Data-driven** - Numbers speak louder than narratives
- **Educational** - Sharing lessons, not selling a system
- **Humble** - Acknowledge losses, iterate publicly
- **No emojis** - Keep it serious and credible

---

## Visual Identity

**Color Palette:**
- Background: `#0a0e27` (deep navy)
- Cards: `#1a2238` (slate)
- Accent Green: `#10b981` (success/profit)
- Accent Red: `#ef4444` (loss/warning)
- Accent Blue: `#3b82f6` (CTA, links)

**Typography:**
- System fonts (San Francisco, Segoe UI, Roboto)
- Clean, readable, modern

**Design Style:**
- Dark theme (crypto-native)
- Minimalist, data-focused
- Mobile-first, responsive
- SVG charts (inline, no dependencies)

## Usage Guidelines

### DO:
- Use "ColdCircuit" as one word, capitalized as shown
- Include tagline "AI-managed crypto, fully transparent." in key materials
- Link to @AICryptoTrades on Twitter
- Use coldcircuit.io (or GitHub Pages URL) in all content
- Maintain dark theme across all visuals
- Keep tone professional and educational

### DON'T:
- Write "Cold Circuit" (two words)
- Use emojis in any content
- Make price predictions or guarantees
- Use hype language ("moon", "to the moon", "100x")
- Promote other trading services
- Claim to be financial advice

## FAQ Responses

**Q: Is this financial advice?**
A: No. This is an experiment and learning journey. I'm sharing data, not recommendations.

**Q: Can I copy your strategy?**
A: The code is open. Strategy is documented. Use at your own risk.

**Q: What's your win rate?**
A: Check the live dashboard at coldcircuit.io. All metrics are public.

**Q: Why such a small position?**
A: This is a learning experiment. Small stakes allow me to validate the strategy before scaling.

**Q: What platform do you use?**
A: Coinbase Advanced Trade API. All trades via Python script.

**Q: How often do you trade?**
A: The bot scans every 30 minutes. Trades execute when signals align.

## File Naming Conventions

- Use lowercase with hyphens: `trade-log.html`, `blog-updater.py`
- Prefix date-specific files: `daily-2026-03-02.txt`, `weekly-summary-2026-03-02.txt`
- Keep URLs clean: `/index.html`, `/strategy.html` (not `/page1.html`)
